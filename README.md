# 🎮 Hungama Game

A fun and interactive Python-based game developed as part of my internship project at CodeAlpha. The game is designed to provide an engaging gaming experience while demonstrating core Python programming concepts such as loops, functions, conditional statements, and user interaction.

## 📌 Features

- Interactive command-line gameplay
- User-friendly interface
- Score tracking system
- Randomized game elements for replayability
- Lightweight and easy to run
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3.x
- Random Module
- Standard Python Libraries
